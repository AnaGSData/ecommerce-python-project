# E-commerce Python Project

Python project focused on applying programming fundamentals to an online store business case.

This project was developed as part of the **Adalab Data Analytics & AI Bootcamp** and has been adapted for portfolio purposes.

---

## Overview

The goal of this project was to apply core Python concepts through a practical business-oriented scenario based on an online store.

The project focuses on using Python to manage and structure information related to products, customers and store operations, while practicing programming logic, problem-solving and code organization.

---

## Objective

The main objective was to use Python to solve practical exercises related to an e-commerce environment.

The project includes practice with:

- Variables and data types
- Lists and dictionaries
- Conditional statements
- Loops
- Functions
- Classes and methods
- Data structure manipulation
- Business-oriented programming logic
- Code organization and readability

---

## Business Context

The project simulates an online store environment where Python is used to manage products, customers and basic store operations.

This type of exercise helps connect programming fundamentals with real-world business logic, such as organizing product information, managing customer data, applying simple calculations or automating store-related processes.

## Project Workflow

### 1. Problem Understanding

Each exercise was reviewed to understand the expected logic, inputs and outputs within the online store context.

### 2. Python Implementation

The solutions were implemented using core Python concepts, focusing on clarity, readability and correct logic.

### 3. Bonus Challenge

An additional bonus section was completed in a separate Jupyter Notebook to extend the original assessment requirements and practice more advanced problem-solving.

### 4. Code Review and Testing

The code was reviewed and tested to check that the expected outputs were obtained.
